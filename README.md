# Mini Streaming Platform

A minimal video streaming service built with HLS and adaptive bitrate.

## Features

- HLS streaming
- FFmpeg transcoding
- Vue video player
- Docker environment

## Tech Stack

- Vue
- Node.js (Express)
- FFmpeg
- HLS
- Docker

## Run

```bash
docker compose up --build
```
